# Examples

Runnable examples that demonstrate Lythra's key features. Each example can be run directly:

```bash
GEMINI_API_KEY="..." node ./bin/lythra run examples/<file>.lth
```

## Quick Reference

| Example | Features Demonstrated |
|---|---|
| [Email Classifier](./email-classifier) | `for` loops, `vision`, `precise`, `attempt`/`fallback` |
| [Content Pipeline](./content-pipeline) | `pipeline`, `parallel`, `attempt`, `fuzzy` |
| [AI Web Server](./ai-server) | `server`, `channel`, `filter`, `transmit`/`receive` |
| [CLI Chatbot](./chatbot) | `while` loop, `vision`, conversation history |
